A. English
----------
1.- Install the program by uncompressing the Zip file to a new subdirectory (folder) in your hard drive. Make sure you have full-control access in that subdirectory or SO will not be able to save your team's progress properly.
2.- Install JRE (Java Runtime Environment) version 1.5 (aka Java 5) or later if you don't have it in your system. You can get it from http://java.sun.com/javase/downloads/
3.- Run the program by double clicking on "so.jar" . If your system is properly configured this shold start the JVM and the program automatically. If NOT: open a console window, navigate to the directory where you installed the program and then type "java -jar so.jar".

Enjoy!


** IMPORTANT **
The installation path cannot have special characters such as [ ]
i.e. this path is valid:
D:\My documents\Sokker

this is NOT:
D:\test[123]\sokker

3.- Ejecuta el programa haciendo doble-click en "so.jar". Si tu sistema está correctamente configurado esto debería iniciar automáticamente la JVM y el programa. En caso CONTRARIO: abre una ventana de consola, dirígete al directorio donde instalaste el programa y tipea "java -jar so.jar".

Disfrútalo! ;)

C. Deutsch
----------
1.- Installiere das Programm, indem du den ZIP-Ordner in ein neues (Unter-)Verzeichnis entpackst (Rechtsklick: "Alle extrahieren..."). Vergewissere dich, dass du uneingeschränkten Zugriff hast zu dem Ordner, in welchen du die ZIP-Datei extrahiert hast. Ansonsten wirst du eventuell nicht in der Lage sein, die Fortschritte deines Teams richtig abzuspeichern.
2.- Installiere das JRE (Java Runtime Environment) Version 1.5 (aka Java 5) oder eine neuere. Wenn du Java 5 oder höher nicht auf deinem Computer installiert hast, kannst du es von http://java.sun.com/javase/downloads/ kostenlos herunterladen.
3.- Starte das Programm (SO), indem du auf die Datei "so.jar" doppelklickst. Wenn du alles richtig installiert hast, sollten das JRE und der SO automatisch starten. Falls dies nicht der Fall ist: Klicke mit der rechten Maustaste auf SO und dann auf "Öffnen mit..." und wähle dort "java - jar so.jar" aus. 

Viel Spass!


** Wichtig **
Der Installationspfad darf gewisse Zeichen wie diese: [ ] nicht haben.
D.h.: Dieser Installationspfad ist zulässig:
D:\My documents\Sokker

Dieser hier ist es NICHT:
D:\test[123]\sokker

D. Romana
----------
1.- Instaleaza programul prin decomprimarea fisierului Zip într-un director la tine pe calculator. Trebuie sa fi sigur ca ai drepturi de scriere/citire in acel director altfel nu vei putea salva progresul echipei tale.
2.- Instaleaza JRE (Java Runtime Environment) version 1.5 (alias Java 5) versiuni superioare (daca nu exista deja in sistemul tau de operare). Il poti lua de la adresa http://java.sun.com/javase/downloads/
3.- Ruleaza programul prin dublu click pe "so.jar" . Daca sistemul tau este bine configurat aceasta operatie va porni JVM (JAVA VIRTUAL MACHINE) si programul automat. Daca sistemul nu e bine configurat: creaza* un fisierul start.bat, in directorul SO (trebuie sa aiba aceeasi cale cu SO.jar), in care se va scrie urmatoarea linie: java -jar so.jar
Acum deschide start.bat si programul va porni.
*Daca folosesti Total Commander pentru a creea un fisier nou apasa Shift+F4
Savureaza!


** IMPORTANT **
Calea de instalare nu poate cuprinde caractere speciale ca [ ]
adica aceasta cale este valida:
D:\My documents\Sokker

aceasta NU este:
D:\test[123]\sokker
